Dimpact Zaakafhandelcomponent (ZAC)

This repository contains the source code of the "zaakafhandelcomponent" (ZAC) developed for Dimpact. The zaakafhandelcomponent is an open-source, generic, workflow-based component for managing 'zaken' in the context of zaakgericht werken, a Dutch approach to case management. It is developed for Dimpact and used by Dutch municipalities. ZAC was initially developed by Atos. Starting July 2023 the development of ZAC was taken over by Lifely and INFO, where INFO is a partner of Lifely. Please refer to GitHub repository for details.

End-to-end testing

On a scheduled basis, we run end-to-end tests on a the Lifely Test Environment. The results are stored in github, and can be found there. The latest end-to-end test results are available here.
